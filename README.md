## 🚀 Sobre mí

Soy el fundador de **PixelForge**, una agencia de diseño y desarrollo web basada en Buenos Aires. Me encargo del desarrollo, el diseño, el marketing y la captación de clientes — todo bajo el mismo techo.

Me especializo en llevar negocios locales al mundo digital: desde la web hasta la automatización de procesos internos.

- 🏙️ Buenos Aires, Argentina
- 🔭 Actualmente construyendo plataformas web para pymes locales
- ⚡ Automatizo procesos con **n8n** para escalar sin contratar
- 📦 Publico infoproductos sobre desarrollo web en Hotmart y Gumroad
